# MediaMeta Specification Charter

The purpose of the MediaMeta Specification is to define an open, implementation-independent specification for describing and exchanging structured metadata about digital media.

The specification exists to promote interoperability, encourage independent implementations, and enable reusable media analysis and media transformation across diverse software ecosystems.

**Understanding media and transforming media are distinct concerns.**

MediaMeta is founded on this principle.

Analysis produces an objective, structured description of media. Transformation consumes that description and applies user-defined policies to produce new media assets. By separating observation from policy, MediaMeta enables independent tools to interoperate through a common specification without prescribing how media should be interpreted or modified.

The MediaMeta Specification is developed as an open community effort and is intended to remain vendor-neutral, implementation-independent, extensible, and openly implementable.
